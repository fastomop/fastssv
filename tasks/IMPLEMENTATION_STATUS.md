Works in WHERE clauses, JOIN ON clauses, HAVING clauses, CASE WHEN expressions, CTEs, and subqueries. Tests: TestConceptNameWhitespace (30 tests covering = operator, IN clauses, JOIN conditions, all TRIM variants, LIKE operators, inequality operators, edge cases with qualified/unqualified columns, and complex queries).*
